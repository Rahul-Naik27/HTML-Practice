# README.txt

Project Name:
HTML Tables Practice

Description:
This project is a collection of practice examples demonstrating how to create and style HTML tables.
Different files are used to experiment with various table features such as borders, spacing, headers, row and column spans, zebra striping, hover effects, and column grouping.
The goal of this project is to understand the structure and styling of tables in HTML using both internal and external CSS.

Files in the Project:
1table.html
Contains a basic HTML table with employee information.
Different CSS files can be linked (by uncommenting them) to observe how different styles affect the table.

2border.css
Demonstrates how to apply borders to tables, rows, and cells.
It also explores different border styles and the use of the `border-collapse` property.

3sizes.css
Shows how to control table size and spacing using properties such as:
* width
* padding
* border-spacing

4headers.html
Demonstrates how table headers work and how to use:
* `<caption>`
* `colspan`
* `rowspan`
This example shows how table cells can span multiple columns or rows.

5colrowspan.html
Shows another example of table headers and demonstrates vertical headers where the header cells appear on the left side of the table.

6tablestyle.css
Contains different styling experiments for tables, including:
* Zebra striping (alternating row colors)
* Vertical zebra striping
* Horizontal dividers
* Hover effects for table rows

These styles help improve table readability and user interaction.
7colgroup.html
Demonstrates the use of the `<colgroup>` and `<col>` elements to apply styles to specific columns in a table.
It also shows how columns can be hidden using the `visibility: collapse` property.

Concepts Practiced:
* HTML table structure (`table`, `tr`, `th`, `td`)
* Table captions
* Table borders and border styles
* Padding and spacing inside tables
* Column span (`colspan`)
* Row span (`rowspan`)
* Zebra striping using CSS
* Hover effects for table rows
* Column grouping with `<colgroup>`

Technologies Used:
HTML5
CSS3

How to Run the Project:
1. Download or clone the project folder.
2. Open any HTML file in a web browser.
3. To test different table styles in `1table.html`, uncomment one of the CSS links in the `<head>` section.

Example:
<link rel="stylesheet" href="border.css">

Notes:
This project is intended for learning and practicing HTML tables and CSS styling techniques.
Each file focuses on a specific table feature to help understand how tables behave in different situations.

Purpose:
Created as part of HTML practice exercises to learn table structure and styling.
