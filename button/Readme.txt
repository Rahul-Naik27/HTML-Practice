Project Name:
Simple HTML & CSS Button Practice

Description:
This is a simple beginner-level web project that demonstrates how to create a styled button using HTML and CSS. The button is created using an anchor (<a>) tag and styled with CSS to look like a clickable button. When the button is clicked, it opens the YouTube website in a new browser tab.

Purpose:
The purpose of this project is to practice basic web development concepts such as:
- HTML page structure
- Linking external CSS
- Styling links to look like buttons
- Using hover effects
- Using basic CSS properties like padding, margin, border, and border-radius

Files in the Project:

1. index.html
   Contains the structure of the webpage including:
   - Page heading
   - Description text
   - A button that links to YouTube

2. btnstyle.css
   Contains all the styling for the webpage including:
   - Resetting default margins and padding
   - Heading styling
   - Paragraph styling
   - Button styling using link selectors
   - Hover effects for the button

Features:
- Styled clickable button
- Hover effect that changes color and border radius
- Clean layout with margins and padding
- External CSS file for styling
- Opens YouTube in a new browser tab using target="_blank"

Technologies Used:
- HTML5
- CSS3

How to Run the Project:
1. Download or copy the project files.
2. Make sure the following files are in the same folder:
   - index.html
   - btnstyle.css
3. Open the "index.html" file in any web browser.
4. Click the "Open YouTube" button to test the functionality.

Purpose:
Created as a practice project for learning basic HTML and CSS.

Notes:
This project is meant for learning and practicing basic front-end web development skills.
