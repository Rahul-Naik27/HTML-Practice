# README.txt

Project Name:
User Information Form (HTML Practice)

Description:
This project is a practice example demonstrating how to create a user input form using HTML. The form collects basic personal information such as name, email, mobile number, gender, age, and a short bio. It also includes interactive elements like a rating slider and a confirmation checkbox.
The goal of this project is to understand how HTML forms work and how different input elements are used to collect data from users.

## Features:
* Structured form using <form>, <fieldset>, and <legend>
* Text input fields for first name and last name
* Email input with built-in validation
* Mobile number input with pattern validation
* Radio buttons for gender selection
* Dropdown menu for selecting age group
* Textarea for writing a short bio
* Range slider to rate the form
* Confirmation checkbox before submitting
* Submit and Reset buttons

## Concepts Practiced:
* HTML form structure
* Label and input association using the "for" and "id" attributes
* Input validation using attributes such as:
  * required
  * pattern
* Different HTML input types:
  * text
  * email
  * tel
  * radio
  * checkbox
  * range
* Dropdown selection using <select> and <option>
* Multiline text input using <textarea>
* Form grouping using <fieldset> and <legend>
* Form submission buttons

## Technologies Used:
HTML5

## How to Run the Project:
1. Download or clone the project folder.
2. Open the HTML file in any web browser.
3. Fill in the form fields and submit the form to test the input elements.

## Purpose:
This project is created for learning and practicing HTML forms. It helps in understanding how web pages collect user data through form inputs and how validation works in HTML.
