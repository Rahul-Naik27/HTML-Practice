# README.txt

Project Name:
HTML Lists Practice

Description:
This project contains several examples demonstrating how different types of lists work in HTML. The examples show how to create unordered lists, ordered lists, nested lists, description lists, and how lists can be styled using CSS. It also includes a simple navigation bar created using an unordered list.

The goal of this project is to understand the structure, behavior, and styling of HTML lists.

Files in the Project:
## unorder.html
This file demonstrates how to create unordered lists using the <ul> and <li> elements.
Concepts shown in this file:
* Default unordered list (disc style)
* List style types such as circle and square
* Removing list markers using "list-style-type: none"
* Creating nested lists (lists inside lists)

## order.html
This file demonstrates how ordered lists work using the <ol> and <li> elements.
Concepts shown in this file:
* Default numeric ordered lists
* Alphabetic lists (uppercase and lowercase)
* Roman numeral lists (uppercase and lowercase)
* Controlling the starting number using the "start" attribute

## otherlist.html
This file demonstrates description lists.
Concepts shown in this file:
* Description list structure using:
  <dl> – description list
  <dt> – term
  <dd> – description
This type of list is useful for definitions, glossaries, and explanatory content.

## navbar.html
This file demonstrates how unordered lists can be used to create a simple navigation menu using CSS.
Concepts shown in this file:
* Removing default list styling
* Horizontal navigation layout using float
* Styling links inside list items
* Hover effects for navigation links
* Creating a simple website navigation bar

## Concepts Practiced:
* Unordered lists (<ul>)
* Ordered lists (<ol>)
* Description lists (<dl>)
* Nested lists
* List styling using CSS
* List style types
* Navigation menus using lists

## Technologies Used:
HTML5
CSS3

## How to Run the Project:
1. Download or clone the project folder.
2. Open any of the HTML files in a web browser.
3. Each file demonstrates a different type of list or list styling technique.

## Purpose:
This project is created for learning and practicing HTML list structures and basic CSS styling. It helps in understanding how lists are used in real web pages, including navigation menus.
